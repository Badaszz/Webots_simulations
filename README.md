# Webots_simulations

This repository contains two Webots simulation projects for mobile robot behavior and localization.

## Projects

- `line_following_obstacle_avoidance`
  - A Webots project that demonstrates line following combined with obstacle avoidance for an e-puck robot.
  - Includes a Python controller and a world file with floor sensors for line detection.

- `localization_with_kalman_filter`
  - A Webots project focused on robot localization using odometry and a Kalman filter.
  - Includes Python scripts for basic localization and Kalman-filter-based estimation, plus a supporting world file.
